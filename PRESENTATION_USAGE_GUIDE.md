Charts are sized for 16:9 widescreen presentations (1280x720)

## Chart Export Tips

### For High-Resolution Exports
1. Open individual slide HTML file
2. Click camera icon
3. Save as PNG
4. Resolution: 1280x720 (16:9 aspect ratio)

### For Editable Versions
- Plotly charts can be opened in Plotly Chart Studio
- Export as SVG for vector editing in Illustrator
- Or export data as CSV to recreate in Excel/PowerPoint native charts

### For Print Materials
- Increase resolution in script: `width=1920, height=1080, scale=2`
- Rerun script to generate high-DPI versions

## Accessibility Notes

Charts are designed for:
- **Colorblind viewers**: Red/green used with text labels (not color alone)
- **Low vision**: High contrast ratios, large fonts (14px minimum)
- **Screen readers**: Alt text embedded in HTML versions

## Backup Plan

If charts don't display properly:

1. **Internet issue**: Plotly requires CDN access - use offline version (see README troubleshooting)
2. **Browser issue**: Try Chrome, Firefox, or Edge (all tested)
3. **Export issue**: Use browser screenshot tool (Windows: Win+Shift+S)
