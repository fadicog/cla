## Export to PowerPoint (3 Steps)

### Step 1: Open Combined Report
Double-click `executive_presentation_visuals.html`

### Step 2: Export Each Chart
- Hover over chart
- Click camera icon (top-right corner)
- Download PNG

### Step 3: Insert into PowerPoint
- Create 16:9 widescreen deck
- Insert → Picture → Select downloaded PNG
- Resize to fit slide

**Done!** You have 7 executive-quality slides ready.

## Troubleshooting

**Charts not showing?**
- Check internet connection (Plotly needs CDN access)
- Try different browser (Chrome recommended)

**Export not working?**
- Right-click chart → "Save image as..."
- Or use browser screenshot (Win+Shift+S)

**Need to update data?**
- Edit `executive_presentation_visuals.py`
- Rerun: `python executive_presentation_visuals.py`
